# Sketches: Dashboard Finance Cards

Desain visual card statistik keuangan di dashboard. Kami membandingkan dua pendekatan:

1. **Variant A: Vercel White** (Terang, Airy, menggunakan shadow-as-borders).
2. **Variant B: Linear Dark** (Gelap, Sleek, dengan glow neon minimalis).

## Head-to-Head Comparison

| Kriteria | Variant A (Vercel White) | Variant B (Linear Dark) |
|---|---|---|
| **Latar Belakang** | Terang (`#ffffff` & `#fafafa`) | Gelap Gulita (`#0c0d0e`) |
| **Gaya Border** | Shadow 1px spread tanpa border CSS nyata | Border solid semi-transparan `rgba(255,255,255,0.08)` |
| **Hover Effect** | Peningkatan bayangan halus | Glow warna ungu (`#5e6ad2`) |
| **Typography** | Geist Sans + Geist Mono | Geist Sans + Geist Mono |
| **Vibe** | Bersih, profesional, aman | Modern, canggih, dev-centric |
